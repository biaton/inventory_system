AIM WMS (Asia Integrated Machine Inc. Warehouse Management System)

An enterprise-grade, full-stack Warehouse Management System (WMS) designed for 24/7 logistics operations. Built with a focus on data density, real-time alerts, AI-driven slotting, and 3D spatial visualization.

OVERVIEW
AIM WMS is a comprehensive inventory and logistics command center. It covers the entire supply chain lifecycle—from Procurement (PO) and Inbound Logistics, to Smart Order Picking, Fleet Dispatch, and Reverse Logistics (RMA).

KEY FEATURES

1. Advanced Dashboards and Analytics
Hybrid Command Center: Draggable smart capsule UI with Day (Sky) and Night (Space) modes.
Analytics Board: Real-time KPI cards for Inventory Value, Critical Stocks, and Capacity Utilization.
Smart Purchasing Advisor (AI Forecast): Calculates Average Daily Consumption (ADC) and Days of Supply (DOS) to predict depletion and suggest re-order quantities.
System Health Audit: Automated batch engine that scans 7 critical data points (Expiry, Low Stock, Overdue Logistics, QC Backlogs, etc.) and dispatches daily digests.

2. Core Logistics and Fulfillment
Procurement: Multi-supplier batching and authorization queues.
Inbound and Receiving: Advance Shipping Notice (ASN), Dock Arrival Verification, and Cross-Dock Auto-Interception logic.
Outbound and Sales: Manual/Bulk Excel order creation, Smart Order Picking (enforcing strict FIFO), and e-POD (Electronic Proof of Delivery).
Reverse Logistics (RMA): Handling customer/supplier returns with Quarantine routing and Smart Auto-Replacement synergy.

3. Administration and Security
Role-Based Access Control (ACL): Granular modular permissions for different employee roles.
Notification Routing: Configurable matrix for Email and In-App Web Alerts.
System Audit Logs: Immutable historical tracking of every transaction (REG, OUT, MOVE, ADJ) including IP Address logs.

TECHNOLOGY STACK
Backend: Python, Django
Frontend: HTML5, Tailwind CSS, JavaScript (Vanilla)
Icons: Lucide Icons
